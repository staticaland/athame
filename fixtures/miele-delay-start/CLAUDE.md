# Technical Setup

Calculates Miele W1 washing machine delay start times. Single-page app, vanilla JavaScript.

## Stack

- Vanilla JS (ES modules)
- Tailwind CSS v4 + PostCSS
- Vite v5
- Vitest
- Docker → nginx → Fly.io

## Files

- `index.html` - UI
- `script.js` - Logic
- `script.test.js` - Tests
- `dist/` - Build output
